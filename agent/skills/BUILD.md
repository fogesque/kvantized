# Инструменты разработки

Команда использует следующий набор инструментов для разработки:

- **Язык:** C++23
- **Компиляторы:**
  - Linux: GCC 14
  - Windows: MSBuild
- **Система сборки:** CMake с файлом `CMakePresets.json`
- **Генератор сборки:** 
  - Ninja (Linux)
  - Visual Studio (Windows) 
- **Менеджер пакетов:** vcpkg
- **Форматирование кода:** clang-format
- **Статический анализ:** clang-tidy
- **Развертывание:** Docker
